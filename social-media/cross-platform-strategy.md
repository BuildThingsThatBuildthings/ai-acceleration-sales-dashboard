# Cross-Platform Content Strategy

One visual asset. Platform-specific captions. Maximum efficiency.

---

## The Core Principle

**Create once. Post everywhere. Customize captions.**

Every piece of visual content (video or image) can work across all platforms. What changes is:
- Caption length and style
- Hashtag strategy
- Call to action
- Posting format

## Platform-Specific Guidelines

### TikTok Captions

| Element | Guideline |
|---------|-----------|
| Length | 1-2 sentences max |
| Tone | Casual, punchy |
| Hashtags | 3-5, trending + niche |
| CTA | Implied in video, minimal in caption |
| Emoji | 1-2 maximum |

### Instagram Reels/Feed

| Element | Guideline |
|---------|-----------|
| Length | 3-7 sentences |
| Tone | Professional but approachable |
| Hashtags | 3-5 at end |
| CTA | Save, comment, follow |
| Emoji | Strategic, not excessive |

### YouTube Shorts

| Element | Guideline |
|---------|-----------|
| Length | 2-4 sentences + links |
| Tone | Informative |
| Hashtags | 3-5, include #shorts |
| CTA | Subscribe, link to full content |
| Description | Expand on video value |

### Facebook

| Element | Guideline |
|---------|-----------|
| Length | Full paragraph, 50-150 words |
| Tone | Conversational, story-driven |
| Hashtags | 2-3 maximum |
| CTA | Question to spark discussion |
| Links | In first comment, NOT main post |

### LinkedIn

| Element | Guideline |
|---------|-----------|
| Length | Longest of all platforms, 100-250 words |
| Tone | Professional thought leadership |
| Hashtags | 3-5 at end |
| CTA | Question, "Comment [WORD]" |
| Structure | Short paragraphs, whitespace |

## Batch Posting Workflow

### Step 1: Create Asset (Once)

Film or generate your visual content. Save with proper naming convention.

### Step 2: Write All Captions (Once)

For each asset, write all 5 platform captions at once. Use the templates above.

**AI Prompt for Caption Batch:**

```
I have a video/image about: [TOPIC/DESCRIPTION]

Write platform-specific captions for:

1. TikTok (1-2 sentences, casual, 3-5 hashtags)
2. Instagram Reels (3-7 sentences, save CTA, 3-5 hashtags)
3. YouTube Shorts (2-4 sentences with subscribe CTA, 3-5 hashtags)
4. Facebook (50-150 words, question at end, 2-3 hashtags)
5. LinkedIn (100-250 words, thought leadership tone, 3-5 hashtags)

Brand voice: Professional but approachable. Educational. Direct.
Business: Real estate AI training for brokerages.
```

### Step 3: Schedule (Once per week)

Use scheduling tools to post across platforms:

| Tool | Platforms |
|------|-----------|
| Meta Business Suite | Facebook, Instagram |
| TikTok Creator Tools | TikTok |
| YouTube Studio | YouTube |
| LinkedIn Native | LinkedIn |
| Later/Buffer | All (paid) |

## Platform-Specific Considerations

### TikTok
- Remove watermarks before cross-posting elsewhere
- Post first to TikTok for native feel
- Use trending sounds when possible

### Instagram
- No TikTok watermarks (algorithm buries them)
- Vertical performs better than square for Reels
- Carousels get 2x the saves

### YouTube
- Shorts must be under 60 seconds
- Add timestamps to longer content
- Thumbnails matter even for Shorts

### Facebook
- Links in comments, not main post
- Native video preferred over links
- Engagement in first 30 min is critical

### LinkedIn
- Longer captions perform better
- Keywords matter more than hashtags now
- Multi-image posts get highest engagement
