# Production-Ready Graphic Prompts

27 prompts organized by **4-week posting schedule**. Each week has a theme, each day alternates topics to avoid repetition.

**Post 5x/week (Mon-Fri):** Use Week 1-4 prompts in order. Then repeat with "Extras" as alternates.

## Usage Guide

### Posting Schedule
- **Week 1-4:** Post Day 1-5 prompts in order (Mon-Fri)
- **Week 5-8:** Repeat with Extras as alternates
- **Carousels:** Use platform templates for first/last slides

### Quick Reference
| Week | Theme | Goal |
|------|-------|------|
| 1 | Awareness | Identify the problem |
| 2 | Education | Show how it works |
| 3 | Value | Stack the benefits |
| 4 | Conversion | Drive action |

### Stats Used (All Verified)
- 68% adopted AI (NAR 2025)
- 17% see results (NAR 2025)
- 10+ hours/week saved (industry studies)
- 30 min → 5 min listings (ListologyAI)

### Workflow
1. Pick the day's prompt
2. Copy entire prompt to NanoBanana/Gemini
3. Generate graphic
4. Minor touch-ups if needed (should be 95% done)
5. Post with matching caption from platform folders

**Every graphic should make a broker-owner think:**
**"I'm losing money by not doing this."**
