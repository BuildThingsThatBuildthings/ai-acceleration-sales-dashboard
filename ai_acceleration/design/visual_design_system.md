### Typography

```
FONT STACK:

PRIMARY (Headings):
Font: Montserrat
Weights: 700 (Bold), 600 (SemiBold)
Usage: All slide titles, handout headers, section breaks

SECONDARY (Body):
Font: Open Sans
Weights: 400 (Regular), 600 (SemiBold)
Usage: All body text, bullet points, instructions

MONOSPACE (Code/Technical):
Font: Fira Code / Roboto Mono
Weight: 400 (Regular)
Usage: Prompts, code snippets, technical examples

SIZES:
- Slide Title:        48pt (Montserrat Bold)
- Slide Subtitle:     32pt (Montserrat SemiBold)
- Slide Body:         24pt (Open Sans Regular)
- Handout Title:      24pt (Montserrat Bold)
- Handout Section:    18pt (Montserrat SemiBold)
- Handout Body:       12pt (Open Sans Regular)
- Captions/Footnotes: 10pt (Open Sans Regular)
```
