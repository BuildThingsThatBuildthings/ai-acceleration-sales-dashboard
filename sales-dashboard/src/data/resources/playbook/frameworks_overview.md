# Sales Frameworks Overview

Memorize these frameworks. They make every call easier.

---

## CALL STRUCTURE FRAMEWORKS

### PACE (The Call Flow)

Every successful call follows this pattern:

```
┌─────────────────────────────────────────────────────────┐
│  P  │ PATTERN INTERRUPT                                 │
│     │ "I saw [Hook from sheet]..."                      │
│     │ Goal: Get their attention                         │
├─────┼───────────────────────────────────────────────────┤
│  A  │ ASK PERMISSION                                    │
│     │ "Mind if I ask one question?"                     │
│     │ Goal: Get consent to continue                     │
├─────┼───────────────────────────────────────────────────┤
│  C  │ CLARIFY PAIN                                      │
│     │ "What's your biggest bottleneck?"                 │
│     │ Goal: Understand their problem                    │
├─────┼───────────────────────────────────────────────────┤
│  E  │ EXPLAIN VALUE                                     │
│     │ "Here's what we do... [connect to their pain]"    │
│     │ Goal: Show how you solve it                       │
├─────┼───────────────────────────────────────────────────┤
│  →  │ CLOSE                                             │
│     │ "[Day 1] or [Day 2]—which works?"                 │
│     │ Goal: Get the meeting                             │
└─────┴───────────────────────────────────────────────────┘
```

**Remember:** PACE then CLOSE.

---

### HBC (The Opening)

How to start every call:

```
┌────────────────────────────────────────────────────────┐
│  H  │ HOOK                                             │
│     │ Read the AI Opening Hook from the sheet          │
│     │ "Congrats on the record year, Sarah..."          │
├─────┼──────────────────────────────────────────────────┤
│  B  │ BRIDGE                                           │
│     │ Connect to what you do                           │
│     │ "I help brokerages get agents using AI..."       │
├─────┼──────────────────────────────────────────────────┤
│  C  │ CLOSE                                            │
│     │ Ask permission to continue                       │
│     │ "Mind if I ask one question?"                    │
└─────┴──────────────────────────────────────────────────┘
```

**Example in action:**
```
"Hi Sarah, this is John with AI Acceleration.
I saw Semonin had a record year—congrats on leading that team.
I help brokerages get agents using AI that actually saves time.
Mind if I ask one question?"
```

## OBJECTION FRAMEWORKS

### EIR (Handle Any Objection)

```
┌─────────────────────────────────────────────────────────┐
│  E  │ EMPATHIZE                                         │
│     │ "I hear you..." / "That makes sense..."           │
│     │ Don't argue. Acknowledge their concern.           │
├─────┼───────────────────────────────────────────────────┤
│  I  │ ISOLATE                                           │
│     │ "Is it the timing or the investment?"             │
│     │ Find the REAL objection.                          │
├─────┼───────────────────────────────────────────────────┤
│  R  │ REFRAME                                           │
│     │ "Here's another way to look at it..."             │
│     │ Give them a new perspective.                      │
└─────┴───────────────────────────────────────────────────┘
```

**Example:**
```
Them: "It's too expensive."

You: "I hear you—$5K is a real investment.        [EMPATHIZE]
      Is it the total amount or the timing?       [ISOLATE]
      [They say: total amount]
      Here's another way to look at it:           [REFRAME]
      That's $500 per agent. One extra deal
      pays for it twice over."
```

---

### The 3-Layer Response

For tough objections, stack three responses:

```
LAYER 1: Logic (The math)
LAYER 2: Emotion (The story)
LAYER 3: Alternative (The pivot)
```

**Example for "Too Expensive":**
```
LAYER 1: "$500/agent. 12:1 ROI. Pays for itself month one."

LAYER 2: "Your agents are working 60 hours—10 of those are admin.
          What would they do with that time back?"

LAYER 3: "We also have a 3-hour option at $3,000 if that
          works better for your budget."
```

## VALUE FRAMEWORKS

### The Value Stack (3 Layers)

Always pitch in this order:

```
┌─────────────────────────────────────────────────────────┐
│  1  │ TIME                                              │
│     │ "5-10 hours back per week per agent"              │
│     │ This is the hook—everyone wants more time         │
├─────┼───────────────────────────────────────────────────┤
│  2  │ MONEY                                             │
│     │ "$500/month per agent in recovered productivity"  │
│     │ Translate time into dollars                       │
├─────┼───────────────────────────────────────────────────┤
│  3  │ PROTECTION                                        │
│     │ "Fraud training—$500M stolen last year"           │
│     │ Add urgency with risk avoidance                   │
└─────┴───────────────────────────────────────────────────┘
```

### The $500 Math (Memorize Word-for-Word)

```
"If AI saves each agent 5 hours a week,
that's $500 a month per agent.
For a team of [X], that's [X × $500] per month.
Times 12 months, that's [X × $6,000] per year.
The workshop is $5,000.
It pays for itself in month one."
```

**Quick math reference:**
| Agents | Monthly | Annual | ROI |
|--------|---------|--------|-----|
| 5 | $2,500 | $30,000 | 6:1 |
| 10 | $5,000 | $60,000 | 12:1 |
| 20 | $10,000 | $120,000 | 24:1 |

## CLOSING FRAMEWORKS

### The Close Menu

Pick the right close for the situation:

```
┌──────────────┬──────────────────────────────────────────┐
│ DIRECT       │ "Ready to get on the calendar?"          │
│              │ Use when: Strong buying signals          │
├──────────────┼──────────────────────────────────────────┤
│ ALTERNATIVE  │ "[Day 1] or [Day 2]—which works?"        │
│              │ Use when: Need to prompt a decision      │
├──────────────┼──────────────────────────────────────────┤
│ ASSUMPTIVE   │ "How many agents should I expect?"       │
│              │ Use when: They're clearly interested     │
├──────────────┼──────────────────────────────────────────┤
│ SUMMARY      │ "So to recap... worth exploring?"        │
│              │ Use when: Long conversation, need focus  │
├──────────────┼──────────────────────────────────────────┤
│ TRIAL        │ "Does this sound like it could help?"    │
│              │ Use when: Unsure if they're ready        │
└──────────────┴──────────────────────────────────────────┘
```

---

### The ABC Rule

**A**lways **B**e **C**losing... to the next step.

Every call ends with ONE of these:
```
1. A booked meeting (calendar invite sent)
2. A specific follow-up date/time ("I'll call you Tuesday at 2")
3. A clear "no" (mark and move on)
```

Never: "I'll follow up sometime" or "Let me know"

## ACTIVITY FRAMEWORKS

### The 3-3-3 Rule

```
3 hours of calls    = 40 dials minimum
3 outcomes tracked  = Connected / Voicemail / Bad Data
3 daily goals       = 15 conversations, 2 bookings, 5 follow-ups
```

### The Temperature System

```
HOT  = Reply to email, referral, "let's talk"  → CALL NOW
WARM = Opened email 2x, 10+ agents             → CALL TODAY
COLD = No engagement, unknown                  → FILL GAPS
```

## How to Use These Frameworks

### Before Your First Call Day
1. Memorize PACE and HBC
2. Practice the $500 Math out loud 10 times
3. Know the EIR formula

### During Calls
- Follow PACE structure naturally
- Use EIR when you hit objections
- Pick from the Close Menu when ready

### After Each Call
- Did you follow PACE?
- Did you get a next step?
- What framework would have helped?

### Weekly Review
- Which frameworks are you using most?
- Which do you need to practice?
- What patterns are you seeing?

---

## The Golden Rule

**Frameworks are training wheels.**

Use them until they're automatic.
Then they become invisible—
you're just having natural conversations
that happen to follow winning patterns.
