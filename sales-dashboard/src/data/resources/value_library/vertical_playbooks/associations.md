# Vertical Playbook: Associations

## Target Profile

### Decision Makers
| Title | Priority | Notes |
|-------|----------|-------|
| Executive Director | P1 | Budget and strategy authority |
| Education Director | P1 | Owns training programs |
| CEO | P1 | Final approval |
| Education Committee Chair | P2 | Influences decisions |
| Events Manager | P2 | Logistics partner |

### Organization Characteristics
- Local, state, or regional associations
- Member count: 100-5,000+
- Focus on member value and CE credits
- Event-driven revenue model

---

## Pain Points (What Keeps Them Up at Night)

### 1. Member Value Proposition
```
Problem: "Why should I pay dues?"
Why it matters: Declining membership, competition from NAR/online
Our solution: High-value exclusive training that justifies membership
```

### 2. CE Credit Requirements
```
Problem: Members need credits, association needs offerings
Why it matters: CE drives attendance, attendance drives value
Our solution: CE-eligible workshop (check state requirements)
```

### 3. Event Attendance
```
Problem: Hard to get members to show up
Why it matters: Empty rooms = wasted investment
Our solution: Hot topic (AI) drives registrations
```

### 4. Sponsor Revenue
```
Problem: Need vendor sponsors for events
Why it matters: Sponsors offset costs, fund programs
Our solution: Vendor sponsorship model built in
```

### 5. Staying Relevant
```
Problem: Members can get training anywhere
Why it matters: Association must be the go-to resource
Our solution: Cutting-edge AI training shows leadership
```

## Objections Specific to Associations

### "We already have AI on our training calendar"
```
"That's great—what are you offering?
Most association AI training is concepts and overview.
We're implementation—members build live, leave with tools.
Could be a good complement to what you have."
```

### "Our members aren't tech-savvy"
```
"Neither are most agents. That's why we design for
non-technical learners. Mental models, not jargon.
We've trained rooms of agents who'd never used AI—
they leave confident and using it."
```

### "We can't justify the cost"
```
"What if sponsors covered part of it?
Lenders and title companies want member access.
They sponsor, members benefit, association looks good.
Can I walk you through the sponsor model?"
```

### "We need CE credits to drive attendance"
```
"Understood. We can work with you on CE approval.
The curriculum maps well to professional development hours.
What's your state's approval process?"
```

### "We'd need to run it by the education committee"
```
"Of course. What would help you make the case?
I can provide:
- Curriculum overview
- ROI for members
- Sponsor model details
- Attendance projections from similar events

When does the committee meet next?"
```

## Event Format Options

### Half-Day Workshop (3 hours)
```
Format: Morning or afternoon session
Capacity: 20-50 attendees
CE potential: 3 hours
Cost: $3,000
Best for: Regular training calendar, lunchtime events
```

### Full-Day Intensive (6 hours)
```
Format: 9 AM - 4 PM with lunch
Capacity: 20-30 attendees
CE potential: 6 hours
Cost: $5,000
Best for: Signature events, annual conference add-on
```

### Conference Breakout Session
```
Format: 60-90 minute session
Capacity: 50-100 attendees
CE potential: 1-1.5 hours
Cost: Custom (contact us)
Best for: Annual conference programming
```

### Train-the-Trainer
```
Format: Full day + certification
Capacity: 5-10 trainers
CE potential: 6+ hours
Cost: Custom (contact us)
Best for: State associations wanting local delivery
```
